Kenotom Summer Internship 2023: Thessaloniki, Greece

This post explains my summer at Kenotom in Thessaloniki Greece

June - August 2023 David Herrera

Summer of 2023, I completed an internship at Kenotom P.C. in Thessaloniki, Greece. Kenotom is a company focusing on embedded systems for automobiles. My project was in the research and development department, where I worked on creating a framework for the validation of autonomous vehicle software. I worked with a partner, Uzair Malik, and together we used a traffic simular called Carla to create a framework for the validation of autonomous vehicle software.

In order to know which tests needed validation, we created a spreadsheet outlining each test from the United Nation's Autonomous Lane Keeping System Handbook. With this information, an apropriate python script was able to be created to run tests on the Carla simulator. While this project is proprietary to Kenotom, invaluable programming skills, particularly with XOSC file parameterization, were learned.
